# QueryPie、Orca SecurityとともにISEC 2023へ共同出展

> Original source: [TechSeoul News](https://www.techseoul.news/mobile/article.html?no=239)
>
> Imported from syndicated copy: [BoanNews](https://m.boannews.com/html/detail.html?idx=121829)

## 日本語訳

CHEQUERは、2023年9月19日から20日にソウルCOEXで開催されるアジア最大級のセキュリティイベント「ISEC 2023（第17回 International Security Conference）」に参加し、6月にパートナー契約を締結したOrca Securityと共同ブースを運営すると発表しました。

同社は、クラウドデータ保護プラットフォーム（CDPP）としてのQueryPie DACおよびQueryPie SACに、クラウドネイティブアプリケーション保護プラットフォーム（CNAPP）であるOrca Security Premiumを組み合わせた、より強固なトータルサイバーセキュリティサービスを提案する計画です。

Orca SecurityのCNAPPは、CSPM、CWPP、CIEMを含む包括的なクラウド保護を、エージェントレスで実現できる点が特徴とされています。

イベント初日の9月19日には、前年のISEC 2022でBest Speakerに選ばれたCHEQUERのPark Kwan-soon CISOによる講演も予定されており、クラウドネイティブ環境におけるセキュリティとデータガバナンス活用の具体的な解決策が紹介される予定です。

## 原文（韓国語）

체커가 2023년 9월 19일부터 20일까지 서울 코엑스에서 열리는 아시아 최대 보안행사 ISEC 2023(제17회 국제 시큐리티 콘퍼런스)에 참여한다고 밝혔다. 체커는 올해 6월 파트너 계약을 체결한 오르카 시큐리티와 함께 공동 부스를 운영한다.

이번 행사에서 체커는 클라우드 데이터 보호 플랫폼(Cloud Data Protection Platform, CDPP)의 선두주자로 자리매김한 ‘쿼리파이’의 데이터 접근제어 서비스(QueryPie DAC)와 시스템 접근제어 서비스(QueryPie SAC)에, 클라우드 네이티브 애플리케이션 보호 플랫폼(Cloud Native Application Protection Platform, CNAPP)인 Orca Security Premium을 결합한 강화된 토털 사이버보안 서비스를 제안할 계획이다.

에이전트 없이 클라우드 환경 자체에서 운영 관리가 가능한 오르카 시큐리티의 CNAPP는 클라우드 보안 상태 관리(CSPM), 클라우드 워크로드 보호 플랫폼(CWPP), 클라우드 ID 및 권한관리(CIEM)를 아우르는 전방위 클라우드 보호가 가능한 것이 특징이다.

행사 첫째 날인 9월 19일에는 지난해 ISEC 2022에서 ‘최고의 연사(Best Speaker)’에 선정된 체커 박관순 CISO의 클라우드 네이티브 환경 보안 강연도 진행된다. 박관순 CISO는 데이터 거버넌스 솔루션을 통해 기업이 클라우드 환경에서 비용을 절감하고 수익을 증대할 수 있도록 개발 인프라, 데이터 접근 제어 등 관련 업무를 통합 관리할 수 있는 보안 해법을 제시할 예정이다. 또한 쿼리파이와 오르카 시큐리티의 공동 강연도 둘째 날인 9월 20일 진행된다.

체커 황인서 대표는 “이번 ISEC 2023에서 오르카 시큐리티와의 공동 부스 운영을 통해 단일 플랫폼으로 기업의 디지털 자산과 사이버보안에 있어 진일보한 클라우드 보안과 컴플라이언스를 제공할 수 있는 서비스를 알리게 돼 기쁘다”고 밝혔다.

한편 체커는 2023년 6월 오르카 시큐리티와 파트너십 계약을 체결했으며, 국내 클라우드 보안 기술과 글로벌 리딩 보안 솔루션을 결합해 CDPP와 CNAPP를 아우르는 토털 보안 솔루션을 제공하고 있다.
