# ISMSの悪夢？
認定資格の取得のために堂々巡りをしていませんか？この記事で、AWSウェビナーの重要なポイントを整理した内容をご確認ください！クラウド時代の認定資格取得への最短ルートを、ベストプラクティスとともにご紹介します。機知、エネルギー、知識を兼ね備えたCEO の Brant と CIO の Kenny が、ISMS、ISMS-P、PIPA プライバシー法、ゼロトラスト、IAM (Identity and Access Management)、SAML (Security Assertion Markup Language)、SSO (Single Sign-On) について議論します。
