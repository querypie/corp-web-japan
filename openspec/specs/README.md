# OpenSpec specs

| Spec | Type | Purpose |
| --- | --- | --- |
| [`platform-preview-toggle`](./platform-preview-toggle/spec.md) | platform | Defines the non-production Preview Toggle capability for preview navigation, preview-only internal surfaces, and gated-content reviewer unlock behavior. |
| [`platform-component-name-debug`](./platform-component-name-debug/spec.md) | platform | Defines the reusable Component Name Debug marker, mode control, shortcut, overlay, label placement, and Clipboard contract for future implementation. |
| [`contract-deployed-origin-metadata`](./contract-deployed-origin-metadata/spec.md) | contract | Defines that first-party metadata, Open Graph, Twitter image, sitemap, and robots absolute URLs use the currently deployed website origin instead of a hard-coded production origin. |
| [`contract-certifications-content-management`](./contract-certifications-content-management/spec.md) | contract | Defines how acquired-certification information is managed across `/certifications` and supporting static marketing pages while excluding news management. |
| [`contract-global-documentation-sync`](./contract-global-documentation-sync/spec.md) | contract | Defines Global-to-Japan discovery, generation, validation, Draft PR, ignore, notification, runtime, and scheduler behavior. |
| [`contract-lingo-website-migration`](./contract-lingo-website-migration/spec.md) | contract | Defines how the sibling `lingo-web` website is migrated into the `/lingo/**` subtree, including namespace, asset, i18n, scoped CSS, and same-site link contracts. |
