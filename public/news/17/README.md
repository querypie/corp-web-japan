# Localized Screenshot Notes

This directory documents the Japanese-localized derivatives created for the
Lingo and NotePie launch news screenshots.

## Generation Summary

- `lingo-screenshot.ja.png` was generated from
  `lingo-screenshot.png` by localizing only the requested UI regions: the left
  sidebar and the right AI summary panel. The center transcript area was kept
  unchanged.
- `../18/notepie-screenshot.ja.png` was generated from
  `../18/notepie-screenshot.png` by replacing visible Korean UI text across the
  screenshot with Japanese text.
- The localized text was rendered directly onto the screenshots with Japanese
  system fonts, preserving the original screenshots as separate source assets.
