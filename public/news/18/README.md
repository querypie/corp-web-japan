# Localized Screenshot Notes

This directory documents the Japanese-localized derivative created for the
NotePie launch news screenshot.

## Generation Summary

- `notepie-screenshot.ja.png` was generated from
  `notepie-screenshot.png` by replacing visible Korean UI text across the
  screenshot with Japanese text.
- The localized text was rendered directly onto the screenshot with Japanese
  system fonts, preserving the original screenshot as a separate source asset.
